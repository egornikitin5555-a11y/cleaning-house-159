## Инструкция по развертыванию

### 1. Загрузка на хостинг
- Загрузите все файлы в корневую папку вашего хостинга
- Убедитесь что `index.html` находится в корне

### 2. Проверка работы
- Откройте сайт в браузере
- Проверьте hero секцию: видео должно воспроизводиться
- Проверьте адаптивность на разных устройствах
- Убедитесь что параллакс эффект работает

### 3. Настройка
- При необходимости замените видео файл `imgs/hero_video.mp4`
- Обновите контактную информацию в `index.html`
- Настройте формы обратной связи

# Чистый дом 159 - Сайт клининговых услуг

## Описание
Профессиональная уборка квартир и офисов в Перми. Полностью адаптивный сайт с видео фоновым роликом в секции героя.

## Структура проекта
```
CLEANING_HOUSE_STRUCTURED/
├── index.html              # Главная страница
├── css/
│   └── styles.css          # Основные стили (исправленные)
├── js/
│   └── script.js           # JavaScript функциональность
└── media/                  # Медиа файлы
    ├── hero_video.mp4      # Видео для секции героя (1.6MB)
    ├── process_video.mp4   # Видео процесса (1.6MB)
    ├── logo.png            # Логотип компании
    ├── cleaning_bg_*.jpg   # Фоновые изображения
    ├── cleaning_team_*.jpg # Фото команды
    ├── room_cleaning_*.jpg # Фото уборки
    └── image_meta.json     # Метаданные изображений
```

## Развертывание на GitHub Pages

1. **Скачайте все файлы** из архива
2. **Загрузите файлы в GitHub репозиторий** `cleaning-house-159`:
   - `index.html` - в корень репозитория
   - Папку `css/` - полностью
   - Папку `js/` - полностью  
   - Папку `media/` - полностью

3. **Проверьте структуру** в репозитории:
   ```
   repository-root/
   ├── index.html
   ├── css/
   ├── js/
   └── media/
   ```

4. **Включите GitHub Pages** в настройках репозитория:
   - Settings → Pages → Source: Deploy from a branch
   - Branch: main / (root)

5. **URL сайта**: https://egornikitin5555-a11y.github.io/cleaning-house-159/
